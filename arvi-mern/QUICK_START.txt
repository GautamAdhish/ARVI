================================================================================
TROUBLESHOOTING
================================================================================

"npm: command not found"
→ Node.js not installed. Download from https://nodejs.org/

Port 3000 already in use?
→ Stop other npm servers or use: PORT=3001 npm start

Module errors?
→ Delete node_modules folder, delete package-lock.json, run npm install again
